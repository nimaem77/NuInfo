Nu Project Management Office

Nu Project Management Office (Nu PM Office) is a web based tool that can be used to control the simple or complex projects. Nu PM Office is about aligning people, projects and organizational priorities. This tool can help the organizations in efficient and effective management of resources as well as both project-based and ongoing activities. Nu PM Office provides a web interface for all aspects of project management, task creation and assignment, activity reporting and time entry.

Nu Hiring Management System

Hiring Management System is a web based recruitment Management system through which a company fulfills the requirement of its client's. This particular system works around business development officer (BDO), recruiter, employee/consultant and vendor. Recruiters can search the applicant from different sources. The system will have its own Advance search Engine to search for resources from database and resume documents according to requirement. It will have internal communication system to interact among BDO, Recruiter, and employee.

Nu Vendor Management System

Vendor Management System is Web-enabled, customizable secure system to manage vendor relations. It can provide you a single, unified view to understand how your company uses contractors and how much you are spending on them. It can also help you significantly shorten the time it takes to fill a request for a vendor resource. In our Vendor Management the system user can track the status of job orders, contracting process and manage the billing and invoicing.

Nu Contract Management System

Nu Contract Management System (NuCMS) is web enabled, customizable secure system to manage vendor relations. NuCMS provides a user friendly interface between a company and its vendor. It enables the company to quickly respond to fill in job requirements using vendor resources. The vendor can use this system to track the job requests and manage contracts. NuCMS also provides all aspect of project management that includes task creation and assignment, activity reporting and time entry.

Nu Event Management System

Nu Event Management System is a web-based tool that facilitates online registration, payment, resource management, web-based accounting and reporting functionality. This web-enabled, customizable secure system ensures user authentication to access the system. The back office staff can take advantage of an intuitive transaction and reporting interface with minimal training.

Configuration Management Tool

Configuration Management Tool is designed to promote team collaboration while enhancing overall project management and increasing developer productivity. Configuration Management Tool provides all Version Control, Bug Tracking, Change Control, Time and Process Management on it's own or in seamless integration with most of the standard client/server applications.

Trending and Tracking System

Our proposed system will document problems and Process Deviations, the corrective actions that were taken, allow for trending and tracking, and post closure review of problems. This system will also enable Software development and testing teams to coordinate and function more efficiently. For example, the software testing team can enter the discrepancies found during system testing into this software. The software development team can then fix these discrepancies and enter the corrective action taken. This software will facilitate as a database to store information related to the discrepancies and fixes made. The system would allow the managers to trend and track the number of discrepancies and will allow the managers to make the software release decisions.

Dynamic Report Generation Tool

With Dynamic Report Generation Tool from NuIS, users are able to service their own needs by creating, accessing, and distributing reports within an infrastructure that is easily managed and maintained by IT. In this way the IT reporting backlog is virtually eliminated. Dynamic Report Generation Tool allows users to generate reports from any RDBMS using the concepts of ODBC and JDBC. With Dynamic Report Generation Tool you can deliver the power of dynamic reporting across your entire organization: to senior executives, production line workers, to remote sales forces and branch offices around the globe.
